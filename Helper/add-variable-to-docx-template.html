Вернуться
1. создать переменную на форме и в команду на создание дока запихнуть
2. Определить категорию переменной, после чего создать её в nameproject.document_template_variable, где в title указываем имя переменной, которую нужно будет указывать в файле. Имя переменной и параметр на форме должны совпадать
3. по name переменной создать перевод описания в public.string для всех языков, что указаны
4. Добавить в нужные фильтры эту переменную в таблицу nameproject.document_template.use_variable
